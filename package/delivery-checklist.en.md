## Part 2 — Bot Generation (Configurator)

### 2.1 Completing the Configurator Form

- [ ] Open the **Configurator** workflow in n8n
- [ ] Fill in field: `businessName` → ___________________________
- [ ] Fill in field: `businessType` → ___________________________
- [ ] Fill in field: `phoneNumberId` → ___________________________
- [ ] Fill in field: `wabaId` → ___________________________
- [ ] Fill in field: `verifyToken` → ___________________________
- [ ] Fill in field: `notificationTarget` → ___________________________
- [ ] Fill in custom questions (Q2 and Q3) if different from defaults
- [ ] Execute the Configurator

### 2.2 Validating the Generated JSON

- [ ] JSON was generated without execution errors in n8n
- [ ] Open the JSON file and verify:
  - [ ] `businessName` field is correct
  - [ ] `verifyToken` matches the one defined with the client
  - [ ] `phoneNumberId` is correct
  - [ ] Qualification questions are correct and in the right order
  - [ ] Welcome messages include the business name
  - [ ] No access token is hardcoded in the JSON ← critical security check
- [ ] Save the file as: `bot-[business-name].json`
